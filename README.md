# WhatsApp Gateway Backend

Backend API untuk WhatsApp Gateway.

## Deploy ke Railway

1. Fork repository ini
2. Buat project baru di Railway
3. Connect dengan repository GitHub
4. Tambahkan PostgreSQL database
5. Set environment variables

## Environment Variables

```
NODE_ENV=production
JWT_SECRET=random_string_32_karakter
FRONTEND_URL=https://yourdomain.com
```

## Teknologi

- Node.js
- Express
- PostgreSQL
- WhatsApp Web.js
- Socket.IO
